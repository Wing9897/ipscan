# ipscan — 日本語

簡易翻訳版: 基本的な使用法を日本語で提供しています。完全なドキュメントは README.md（English / 繁體中文）を参照してください。

## インストール

```bash
pip install ipscan
```

## 使い方（概要）

- Ping スキャン: `ping_range` または `PingScanner`
- ARP スキャン: `arp_range` または `ArpScanner`

### CLI

```
fping  # 高速連続 ping（対話式）
sping  # ping 範囲スキャン
sarp   # ARP 範囲スキャン
```

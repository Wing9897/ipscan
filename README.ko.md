# ipscan — 한국어

간단한 번역: 기본 설치 및 사용법을 한국어로 제공합니다. 전체 문서는 README.md (English / 繁體中文)를 참조하세요.

## 설치

```bash
pip install ipscan
```

## 사용 개요

- Ping 스캔: `ping_range` 또는 `PingScanner`
- ARP 스캔: `arp_range` 또는 `ArpScanner`

### CLI

```
fping  # 고속 연속 ping (대화형)
sping  # ping 범위 스캔
sarp   # ARP 범위 스캔
```
